# 2048 Game in Java (Console + GUI)

A complete implementation of the classic **2048 game** written entirely in **Java**, featuring both a **console version** and a fully interactive **GUI using Swing**.

This project imitates the original 2048 design with:
- Smooth keyboard controls  
- Color-coded tiles  
- Score tracking  
- Win & Game Over detection  
- Restart functionality  
- Clean and modular code structure  

---

## Features

### Two fully playable versions
- **Console Version (Terminal-based)**
- **GUI Version (Swing-based)** with modern styling

### Game Logic
- Tile merging rules implemented exactly like the original 2048  
- Random tile spawning (2 or 4)  
- Score calculation  
- Win detection at **2048**  
- Full Game Over detection (no moves left)

### GUI Features
- Color-coded tiles  
- Smooth rendering  
- Fullscreen support  
- Restart button  
- Modern “Segoe UI” font  
- Highlight for WIN or GAME OVER states
